##Install of plugins

まずNeoBundle( http://github.com/Shougo/neobundle.vim ) とVimProc
( http://github.com/Shougo/vimproc ) を git clone して,dotvim/vimfiles/bundleに
展開する。
あとはVimを普通に起動して、Install Bundles ? に Yes と答えれば各プラグインがイ
ンストールされる。
